# Assignmen 2

This system is a 3D interactive lighting lab that was created with NumPy, PyOpenGL, and 
Tkinter in Python. It combines an easy-to-use graphical user interface (GUI) with real-time 3D 
rendering to model and show how lighting and material changes affect several 3D objects.
