# Assignment 1

This is a 2D interactive graphics system developed using Python OpenGL. It features 
a real-time analogue clock, an animated MCG logo, and a Malaysian-inspired 
background with Mount Kinabalu and moving elements like clouds, birds, cows, 
and cars.
