# Mini Project
This is a 3D First-Person Shooter (FPS) web game. 
The game was developed using HTML, CSS, and JavaScript with the Three.js library for 3D rendering.
The game comprises 3 level stages, each with increasing difficulty and objectives. Player have to kill the enemy, find the key, and reaching safe zone.

- User Interface (HTML/CSS): Responsible for layout, menus, and buttons.
- Game Engine (JavaScript/Three.js): Core logic and 3D rendering engine.
- Scene & Object Manager: Handles all 3D assets, enemies, keys, and environmental logic.
- Input Handler: Processes player input like movement and interaction.
- Collision & Physics: Detects collision with environment, objects, or enemies.
- Level Manager: Manages progression, level state, and difficulty.
- Storyline System: Displays mission briefings and transition cutscenes between levels.
